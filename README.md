# Imperial – E-commerce Demo con PHP

Proyecto demo de una tienda en línea desarrollada con PHP y MySQL, que incluye panel de administración, control de sesiones y funcionalidades clave de un e-commerce.

## Características
- Panel de administración
- Protección de rutas mediante sesiones en PHP
- CRUD de productos
- Filtrado de productos por etiquetas
- Carrito de compras funcional
- Separación de frontend y backend
- Arquitectura modular
- Login/Creacion de cuentas
- Hash de contraseñas

## Autenticación y seguridad
- Uso de PHP Sessions para:
  - Proteger rutas del panel admin
  - Controlar acceso de usuarios autenticados
- Validaciones básicas del lado del servidor

## Tecnologías utilizadas
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

##  Objetivo del proyecto
Este proyecto fue creado con fines demostrativos para:
- Mostrar manejo de backend con PHP
- Implementar lógica de negocio básica de un e-commerce
- Evidenciar control de sesiones y protección de rutas
- Servir como base escalable para proyectos reales
